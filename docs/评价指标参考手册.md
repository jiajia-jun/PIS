# 评价指标参考手册

## 一、分项指标

### 1. 内点率（匹配质量）

**含义**：RANSAC 筛选后，有效匹配点占总匹配数的比例。反映相邻图片之间特征匹配的可靠性。

**公式**：
```
内点率 = Σ(每对 inlier_num) / Σ(每对 total_matches)
```
> `total_matches` = BFMatcher + Lowe's ratio 后的匹配数
> `inlier_num` = RANSAC 筛选后的内点数

**范围**：0 ~ 1，越高越好

---

### 2. 重投影 RMSE（对齐精度）

**含义**：将源图内点经 H 矩阵投影到目标图空间后，与目标图对应点的平均像素误差。反映拼接的几何对齐精度。

**公式**：
```
对每对相邻图片:
  pts1_proj = H @ [src_x, src_y, 1]ᵀ  （齐次坐标投影）
  pts1_proj = pts1_proj / w            （归一化）
  d = ‖pts1_proj - pts2‖               （欧氏距离）
  pair_rmse = √(mean(d²))              （筛选 d < 100px 的点）

重投影RMSE = mean(所有对的 pair_rmse)
评分 = max(0, 1 - RMSE / max(图像对角线 × 0.005, 1))
```

**范围**：0 ~ 1（评分），越高越好

---

### 3. 全景 SSIM（拼接一致性）

**含义**：全景图自身的局部结构连续性。正常区域偏移 1 像素后内容几乎不变 → SSIM 接近 1.0；拼接缝处偏移跨过缝线 → SSIM 骤降。

**公式**：
```
全景图灰度化 → 排除黑边(亮度 ≤ 10)
滑动窗口 32×32，步长 24，遍历有效区域

每窗口做两组 SSIM:
  SSIM_h = ssim(window[:, :-1],  window[:, 1:])   ← 右移 1px，检测垂直缝
  SSIM_v = ssim(window[:-1, :],  window[1:, :])   ← 下移 1px，检测水平缝

全景SSIM     = mean(所有 SSIM_h, SSIM_v)
全景SSIM_min = min(所有 SSIM_h, SSIM_v)
全景SSIM_std = std(所有 SSIM_h, SSIM_v)
```
> 使用 `skimage.metrics.structural_similarity(data_range=255)`

**范围**：0 ~ 1，越高越好

---

### 4. 有效画布占比（画布利用率）

**含义**：全景图中非黑像素占整张图的比例。黑边越多，说明图片排列或 warping 不合理。

**公式**：
```
有效画布占比 = count(灰度 > 10) / 总像素数
空白占比     = 1 - 有效画布占比
```

**范围**：0 ~ 1，越高越好

---

### 5. 清晰度保持率

**含义**：全景图相对于原始输入图片的清晰度保留程度。拼接过程（blending、resize）不应损失过多细节。

**公式**：
```
Laplacian方差 = Var(Laplacian(img))    ← 衡量图像清晰度

全景清晰度   = Laplacian方差(全景图)
原图平均清晰度 = mean(Laplacian方差(每张原图))

清晰度保持率 = 全景清晰度 / 原图平均清晰度
评分         = min(1.0, 清晰度保持率)   ← 上限 1.0（拼接不应比原图更清晰）
```

**范围**：0 ~ 1（评分），越高越好

---

### 6. 运行效率

**含义**：拼接总耗时的相对评分。耗时越短越好，按图片数量缩放预期。

**公式**：
```
评分 = max(0, 1 - 总耗时(秒) / max(图片数量 × 10, 1))
```
> 每张图预算 10 秒。2 张图 20s 内满分，4 张图 40s 内满分。

**范围**：0 ~ 1，越高越好

---

## 二、综合得分

**含义**：六项评分的算术平均，整体衡量拼接质量。

**公式**：
```
综合得分 = mean(内点率评分, 重投影RMSE评分, 全景SSIM, 有效画布占比, 清晰度评分, 运行效率评分)
```
> 六项权重相等，均归一化到 0~1。

**范围**：0 ~ 1，越高越好

---

## 三、辅助指标

| 指标 | 含义 | 公式 |
|------|------|------|
| 总匹配对数 | 所有相邻对 knnMatch 匹配总数 | `Σ(pair.total_matches)` |
| RANSAC内点数 | 通过 Lowe + RANSAC 筛选的内点总数 | `Σ(pair.inlier_num)` |
| 重投影中位数 | 重投影像素误差的中位数 | `mean(每对 median(d))` |
| 重投影P95 | 重投影像素误差的 95 分位 | `mean(每对 P95(d))` |
| 全景SSIM_min | 最差窗口的 SSIM 值 | `min(所有滑动窗口 SSIM)` |
| 全景SSIM_std | 窗口 SSIM 的标准差 | `std(所有滑动窗口 SSIM)` |
| 全景图清晰度方差 | 全景图 Laplacian 方差原始值 | `Var(Laplacian(全景图))` |
| 原图平均清晰度方差 | 源图 Laplacian 方差均值 | `mean(Var(Laplacian(每张原图)))` |
| 空白占比 | 黑边像素占比 | `1 - 有效画布占比` |
